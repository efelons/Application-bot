# Better Application Bot (discord.py v1.7.3)

A self-hostable Discord Application Bot inspired by **Appy**, but with more commands, features, and flexibility.
Built with `discord.py v1.7.3`, `discord-components`, and `aiosqlite`.

---

## âœ¨ Features

* Create unlimited **application forms** (`!createform`, `!addquestion`)
* Users can apply with `!apply <form>`
* Applications saved to **SQLite database**
* Review applications with **buttons** (Accept / Deny)
* Assign a role to accepted applicants automatically
* Cooldowns and reapply settings
* Export applications for record-keeping

## ğŸ“ Example Setup

1. Create a form:

   ```
   !createform staff "Staff Application"
   ```
2. Add questions:

   ```
   !addquestion staff "Why do you want to join?"
   !addquestion staff "Do you have prior experience?"

   ```
